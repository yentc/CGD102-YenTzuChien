輕鬆上手的情緒追蹤網站

每個月總有幾天心情不好，但不知道是什麼原因導致嗎? Mooding 讓您輕鬆紀錄心情，並精準分析出您疑惑許久 的情緒問題。

精選功能

Primary function

快速地創建記錄

快速創建日誌記錄。用户無須敲字，即可紀錄您的日常生活。您也可添加相應的位置，筆記和想法。

分析您的情緒

Mooding的分析引擎幫您找出影響積極和消極情緒的習慣。

精選解壓文章

Mooding整理出眾多情緒排解文章，供您免費閱讀。

線上諮詢

Mooding 線上諮詢讓您無需舟車勞頓，即可在安心的私人空間，進行諮商。

如何開始

Road map

Phase 1

每日心情紀錄

每天輕鬆地紀錄您的日常生活，包括心情，飲食，活動等等.........。

Phase 2

每月數據統計

在月初時會統計出您上個月的情緒報表，讓您的情緒起伏一目瞭然。

Phase 3

情緒了解/理解

如果您有心情排解問題，可以閱讀心情文章，或線上預約諮詢服務。
